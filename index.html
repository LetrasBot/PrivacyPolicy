Política de Privacidade – Letras

A Política de Privacidade do Letras descreve como as informações são coletadas, utilizadas e protegidas enquanto você utiliza o bot. A sua privacidade é prioridade, e todos os dados são tratados com responsabilidade e transparência.

1. Informações Coletadas

O Letras coleta exclusivamente dados fornecidos pela API do Discord, incluindo:

  • ID de usuário;
  • ID de servidor e canais;
  • Informações estritamente necessárias para execução dos comandos.

Nenhum conteúdo de mensagens é armazenado, exceto logs temporários utilizados apenas para depuração e correção de erros.

2. Armazenamento de Dados

O Letras não mantém bancos de dados externos contendo informações pessoais. Logs temporários podem existir durante períodos curtos para fins de diagnóstico, mas são descartados automaticamente.

3. Uso das Informações

Os dados coletados têm como finalidade:

  • Garantir o funcionamento adequado dos comandos;
  • Melhorar funcionalidades do bot;
  • Corrigir erros e evitar abusos de funcionalidades.

As informações não são compartilhadas com terceiros.

4. Segurança

O Letras utiliza práticas seguras no tratamento de dados, limitando o acesso a informações internas e garantindo que nenhum dado sensível seja armazenado ou distribuído.

5. Direitos do Usuário

Você pode solicitar esclarecimentos ou informações sobre como o bot trata seus dados. Todo usuário tem o direito de:

  • Solicitar remoção de dados temporários (quando aplicável);
  • Pedir informações sobre o uso dos dados pela aplicação.

6. Alterações na Política

Esta Política de Privacidade pode ser atualizada a qualquer momento. Mudanças relevantes poderão ser notificadas no servidor de suporte do Letras.

7. Contato

Para dúvidas ou solicitações relacionadas à privacidade, entre em contato via Discord: @letrista

Ao utilizar o bot Letras, você concorda com esta Política de Privacidade.
